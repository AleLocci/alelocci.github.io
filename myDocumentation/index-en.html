Abstract

Machine Learning Test Ontology (MiLTO) aims to model machine learning test procedures and associated data. The main objective is to characterize failures in machine learning tools, especially TensorFlow information about systems and users. This ontology is the result of  doctoral research carried out at IME-USP.
